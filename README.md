# paw-brukerstotte-test

## Arbeidssøkerregisterets brukerstøtte

Dette er kun for testing og eksperimentering.

## Kom i gang

```sh
deno task dev
```

Dette starter utviklingsserveren med hot-reload. Git-hooks for lint og formatering installeres automatisk ved første oppstart.

### Oppdateringer
For å oppdatere til siste versjoner kan du bruke `deno task outdated --update --latest`, da blir det benyttet et 7 dagers cooldown-vindu for nye pakker.
